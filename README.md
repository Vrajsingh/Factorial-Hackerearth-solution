# Factorial-Hackerearth-solution
My Cpp code for the solution of Factorial

#include<iostream>
using namespace std;
int main()
{
    int n,res=1;
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        res = res * i; 
    }
    cout<<res<<endl;
    return 0;
}
